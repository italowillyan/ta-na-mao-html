Sobre o projeto

Este projeto consiste em uma plataforma web desenvolvida como trabalho da disciplina Autoria Web, no ano de 2025. O objetivo foi aplicar, de forma prática, os conceitos fundamentais de desenvolvimento front-end, incluindo estruturação, estilização, interatividade e organização de código. A aplicação simula uma plataforma de aluguel de residências e veículos, com navegação dinâmica, autenticação simulada e gerenciamento local de reservas.

Equipe
  • Italo Willyan — Desenvolvimento das páginas inicial (index), residências e do sistema simulado de login
  • Mosani Rodrigues — Desenvolvimento das páginas de login, reservas e responsável pelo código em JavaScript
  • Nicolas Santos — Desenvolvimento das páginas de veículo, sobre e responsável pelo código em CSS
Tecnologias utilizadas
  • HTML5 — Estruturação das páginas
  • CSS3 — Estilização personalizada
  • JavaScript (Vanilla) — Interatividade e lógica da aplicação
  • Bootstrap 5 — Framework utilizado como base visual e responsiva
  • Menu dinâmico, modal reutilizável e armazenamento local (localStorage)
Estrutura do Projeto

A plataforma é composta pelas seguintes páginas:

  • Index — Página inicial e busca
  • Login — Autenticação simulada do usuário
  • Residências — Listagem e filtro de imóveis
  • Veículos — Listagem e filtro de veículos
  • Minhas Reservas — Visualização e gerenciamento das reservas
  • Sobre — Informações gerais do projeto
Funcionalidades Implementadas
  • Autenticação simulada utilizando localStorage
  • Menu de navegação dinâmico, adaptado ao estado de login
  • Formulários com validação em JavaScript
  • Modal reutilizável para realização de reservas
  • Filtros dinâmicos para residências e veículos
  • Armazenamento e gerenciamento de reservas via localStorage
  • Interface responsiva utilizando Bootstrap
